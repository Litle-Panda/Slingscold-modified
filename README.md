# Slingscold-modified
* Modified Slingscold Launcher just to install without compiling

# Howto Install

###### Download Package
* https://github.com/Litle-Panda/Slingscold-modified
* Open terminal
* $ cd Slingscold-modified
* $ ./install

###### Git Clone
* Open Terminal
* git clone https://github.com/Litle-Panda/Slingscold-modified.git
* $ cd Slingscold-modified
* $ ./install
